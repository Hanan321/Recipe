🍽️ Arabic AI Chef

Arabic AI Chef is an AI-powered Python application that generates traditional dishes from Arab countries.
Users enter an Arabic country, and the system dynamically selects a popular dish and generates a complete recipe using AI.

🚀 Features
🌍 Supports all 22 Arab countries
🤖 AI dynamically selects a traditional dish
🧾 Generates structured recipes including:
Ingredients
Step-by-step instructions
Estimated cost
Plating description
Image prompt
💬 Interactive command-line interface (CLI)
🧠 Uses prompt engineering for controlled AI output
🧠 How It Works
User enters an Arab country
The system validates the input
The AI selects a traditional dish from that country
The AI generates a structured recipe
The result is displayed in a clean format
🛠️ Technologies Used
Python
OpenAI API
Prompt Engineering
Google Colab
